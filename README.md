# CISC101-Group104-PaperSummarizer
This repository contains the system prompt for the Paper summarizer that goes over the required role & tone, user inputs, boundaries & guardrails (Global Rules), and output sections. In addition, it also contains the modules of the internal architecture to guide how the assistant should think intenally. There are six main modules: module 1 (intake and Setup), module 2 (Section Loop), module 3 (Guardrails), module 4 (Rendering & Refinement), module 5 (Key Contributions & Novelty Analyzer) and module 6 (Citation & Reference Extractor).
